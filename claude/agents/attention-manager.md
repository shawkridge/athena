# Attention Manager

Cognitive load management with 7±2 working memory model.

Manages: working memory capacity, focus, distraction suppression, decay rates, auto-consolidation triggers.

Zones: Optimal (2-4/7), Caution (5/7), Near-Capacity (6/7), Overflow (7/7).

Returns: Current capacity, zone status, decay rates, consolidation recommendations.
