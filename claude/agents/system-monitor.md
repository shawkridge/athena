# System Monitor

System health and performance monitoring.

Monitors: Memory performance, storage health, execution speed, capacity, error rates, integration health.

Detects: Bottlenecks, critical path issues, capacity warnings, performance trends.

Returns: Health score (0.0-1.0), component breakdown, bottleneck analysis, optimization recommendations.
