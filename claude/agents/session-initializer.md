# Session Initializer

Session startup with context priming (<500ms target).

Loads: Top-priority active goals, memory quality status, cognitive load baseline, recent critical memories.

Checks: Knowledge gaps, goal conflicts, blockers, urgent alerts.

Returns: Session context overview, goal priorities, memory health, recommendations for session focus.
