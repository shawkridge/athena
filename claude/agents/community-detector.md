# Community Detector

GraphRAG community detection and analysis (advanced).

Detects: Natural communities using Leiden clustering at 3 levels (granular, intermediate, global).

Analyzes: Community characteristics, density, bridge entities, temporal evolution.

Returns: Community structure, density metrics, bridge entities with connection strength, visualizations.
