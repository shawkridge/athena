# Athena Meta-Memory Skill

Understand your memory system health. Track expertise, manage attention, assess confidence in memories.

## What This Skill Does

- **Quality Scores**: Memory usefulness metrics
- **Expertise Tracking**: Your domain expertise
- **Attention Allocation**: Where your focus goes
- **Confidence**: Confidence in memories

## When to Use

- **"How healthy is my memory?"** - Quality scores
- **"What am I expert in?"** - Expertise areas
- **"Where's my focus?"** - Attention distribution
- **"How confident are these memories?"** - Confidence

## Available Tools

### getQualityScores(layer)
Get memory quality metrics.

### getExpertise()
List expertise areas.

### getAttention()
Analyze attention allocation.

### getConfidence(topic)
Get confidence in memory topic.

## Architecture

Integrates with Athena's Layer 6 (Meta-Memory):
- Multi-layer quality scoring
- Expertise tracking
- Cognitive load monitoring
