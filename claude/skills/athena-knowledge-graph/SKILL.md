# Athena Knowledge Graph Skill

Explore 50,000+ entities and relationships. Find what connects to what, analyze concept communities.

## What This Skill Does

- **Query Entities**: Find concepts
- **Find Relationships**: See connections
- **Analyze Communities**: Find concept groups
- **Explore Graph**: Navigate knowledge structure

## When to Use

- **"Find all entities about..."** - Search concepts
- **"What connects to..."** - Find relationships
- **"What concepts relate?"** - Find community
- **"Graph of..."** - Explore knowledge structure

## Available Tools

### queryEntities(search, limit)
Find entities in knowledge graph.

### findRelationships(entity, relationshipType)
Find connections between concepts.

### analyzeCommunities()
Find groups of related concepts.

### exploreGraph(startEntity, depth)
Navigate the knowledge graph.

## Architecture

Integrates with Athena's Layer 5 (Knowledge Graph):
- 50,000+ entities tracked
- Relationship mapping
- Community detection algorithms
