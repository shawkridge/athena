## 5. Backend Implementation Notes

### 5.1 Discovering Operations

Since Athena has 60+ modules, the backend should:

1. **Auto-discover available operations:**
```python
import pkgutil
import athena

# Discover all submodules
modules = []
for importer, modname, ispkg in pkgutil.iter_modules(athena.__path__):
    if ispkg:
        try:
            module = __import__(f'athena.{modname}.operations', fromlist=[''])
            modules.append(modname)
        except ImportError:
            pass  # Module doesn't have operations.py

print(f"Found {len(modules)} modules with operations")
```

2. **Import operations dynamically:**
```python
from athena.skills.operations import list_skills, execute_skill
from athena.research.operations import get_research_tasks
from athena.code_artifact.operations import list_artifacts
# etc.
```

```python
def register_subsystem_routes(app: FastAPI, subsystem: str):

    for name, func in inspect.getmembers(ops_module, inspect.isfunction):
        if name.startswith('get_') or name.startswith('list_'):

```
